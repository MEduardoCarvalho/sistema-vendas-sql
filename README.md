# Sistema de Vendas - Banco de Dados

Projeto de estudo utilizando SQL Server para modelagem e consultas em banco de dados relacional.

## 📌 Funcionalidades
- Cadastro de clientes, produtos e pedidos
- Relacionamento entre tabelas (chaves estrangeiras)
- Consultas com JOIN
- Relatórios de vendas (total por pedido, cliente que mais gastou)

## 🛠️ Tecnologias
- SQL
- Microsoft SQL Server

## 📊 Exemplos de consultas
- Total de pedidos
- Produto mais vendido
- Cliente com maior gasto

## 🚀 Objetivo
Projeto desenvolvido para prática de modelagem de banco de dados e consultas SQL.
